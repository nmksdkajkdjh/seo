## 二、SEO Geo 优化空间

### 2.1 修复 geo-title Edge 引用的 hero 图

**问题**：`geo-title.js` 引用 `hero-tokyo.jpg`、`hero-osaka.jpg` 等，但 `site/images/` 中不存在，会导致 404。

**方案**：
- 方案 A：为 8 地生成/采购 hero 图，放入 `site/images/`
- 方案 B：改为使用已有缩略图，如 `tokyo-kouhaitou-2026-thumb.jpg` 等

### 2.2 新增地域独立页（A 级优先）

| 地域 | 预估搜索量 | 落地页建议 |
|------|------------|------------|
| 北海道 | 中 | `hokkaido-kouhaitou-2026.html` |
| 静岡 | 中 | `shizuoka-kouhaitou-2026.html` |
| 広島 | 中 | `hiroshima-kouhaitou-2026.html` |
| 仙台/宮城 | 中低 | `miyagi-kouhaitou-2026.html` |

复制福岡/神奈川页模板，调整标题、关键词、Schema、内部链接即可。

### 2.3 首页地域区块强化

- 将「地域別 高配当株おすすめ」提升到首屏或热门文章附近
- 增加锚文本长度，如「東京 高配当株おすすめ2026」而不只「東京」
- 可加入简短说明：「お住まいの地域別おすすめ」

### 2.4 Search Console 地域定向

- 在 Search Console 资源设置中确认「目标国家/地区」为日本
- 定期查看「效果」按国家/地区分组，聚焦日本流量

### 2.5 Geo Schema 增强

- 在首页 WebPage Schema 中增加 `about.Place` 覆盖主要目标地域
- 各地域页已有 `geo.region`、`geo.placename`，保持即可

## 三、高效吸流（自然搜索 + 内容）

### 3.1 长尾关键词布局

| 类型 | 示例 |
|------|------|
| 地域+意图 | 東京 NISA おすすめ銘柄 2026、大阪 配当株 初心者 |
| 问题型 | 高配当株 いつ買う、NISA 成長投資枠 おすすめ |
| 比较型 | 高配当株 vs 成長株、NISA vs つみたて |

在现有文章标题、H2、首段自然插入，避免堆砌。

### 3.2 引流路径优化

```
検索 → 地域页/ランキング页 → Hero CTA / 文末 CTA → LINE 登録
```

- 每篇地域页保持 2–3 处明显 LINE CTA
- 可 A/B 测试：「今すぐ無料登録」vs「東京の皆様　今すぐLINE登録」

### 3.3 热门文章卡片位置

- 将地域页（東京、大阪等）尽可能靠前展示
- 与 Search Console 数据结合，点击高的文章优先展示

## 四、竞价排名（Google Ads / 付费搜索）

### 4.1 落地页准备

| 用途 | 建议页面 | 说明 |
|------|----------|------|
| 品牌/通用 | `/` 首页 | 现成，可微调 CTA |
| 地域词 | `/blog/tokyo-kouhaitou-2026.html` 等 | 与广告语一致 |
| 高意图词 | `/blog/nisa-osusume-meigara-2026.html` | NISA、銘柄等 |

要求：标题与广告语一致、首屏有清晰 CTA、加载速度快。

### 4.2 关键词分组建议

| 广告组 | 关键词示例 | 落地页 |
|--------|------------|--------|
| 地域-東京 | 東京 高配当株 おすすめ | tokyo-kouhaitou-2026 |
| 地域-大阪 | 大阪 高配当株、関西 配当株 | osaka-kouhaitou-2026 |
| NISA | NISA おすすめ銘柄 2026 | nisa-osusume-meigara-2026 |
| 通用 | 高配当株 おすすめ、株 推奨銘柄 無料 | 首页或 2026-kouhaitou-kabuka-ranking |

### 4.3 转化追踪

- **Google Ads 转化**：LINE 登録成功页设置转化（需有单独 thank-you 页或通过 GTM 检测跳转）
- **GA4 事件**：已有 `line_register_click`，可设为 GA4 转化并导入 Ads
- **UTM 参数**：广告链接加 `utm_source=google&utm_medium=cpc&utm_campaign=kouhaitou_geo`

### 4.4 广告文案示例

```
标题1: 東京 高配当株 2026｜プロ厳選
标题2: 毎週無料LINE配信｜32,000人登録
标题3: NISA対応・利回り4%以上

描述: 元証券アナリストが厳選。東京・首都圏向け高配当株TOP10を毎週無料でお届け。登録無料・ブロック1秒。
```
