# kouhaitou.jp 长期修复优化方案

> 6–12 个月可执行的持续优化路线图

---

## 一、方案概览

| 阶段 | 时间 | 重点 | 可量化目标 |
|------|------|------|------------|
| 稳定期 | 第 1–4 周 | 技术修复、索引验证、内链优化 | 索引≥38、LCP<3s |
| 扩展期 | 第 2–3 月 | 主题集群、内容扩展、行业页释放 | 页面≥46、周更稳定 |
| 外链期 | 第 4–6 月 | 外链建设、深度内容、地域扩展 | 外链 20–50、流量月增 |
| 规模期 | 第 7–12 月 | 行业×地域矩阵、FAQ 扩量、持续迭代 | 页面 60+、主词进 Top20 |

---

## 二、技术优化（持续）

### 2.1 每季度执行

| 项目 | 动作 | 负责人 |
|------|------|--------|
| Core Web Vitals | PageSpeed Insights 检测，LCP/CLS/INP 达标 | 技术 |
| Sitemap | 新页及时加入，lastmod 更新 | 内容 |
| 404 检查 | GSC「カバレッジ」排查无效 URL | 技术 |
| 移动端 | 实机测试主要页面 | 技术 |
| 结构化数据 | Schema 校验无错误，新页补全 | 内容 |

### 2.2 新增页面规范

```
✅ 必做：
- canonical 设置
- title 60 字内、description 120–155 字
- H1 与 title 一致
- 至少 4 条内链（1 支柱 + 2 专题/地域 + 1 相关）
- 图片 alt、width/height、loading="lazy"
- sitemap 追加
- lastmod 使用发布日期
```

### 2.3 性能优化路线

| 优先级 | 项目 | 预计收益 |
|--------|------|----------|
| P1 | 图片 WebP/AVIF，压缩至 <200KB | LCP -0.3s |
| P2 | Tailwind 自托管或 critical CSS | FCP -0.2s |
| P3 | 第三方脚本延迟加载（GTM 除外） | TTI 改善 |
| P4 | CDN 静态资源缓存 | 回访用户加载加速 |

## 三、内容优化（持续）

### 3.1 发布节奏

| 周期 | 动作 |
|------|------|
| 每周 | 1 篇新博客（地域 / 专题 / 行业） |
| 每月 | 1 篇深度文（3000 字+）或比较系 |
| 每季 | 利回り・銘柄リスト等时效页更新 |
| 每年 | 全站 2026→2027 年份更新 |

### 3.2 主题集群维护

**支柱页**：首页、2026-kouhaitou-kabuka-ranking、nisa-osusume-meigara-2026

**每篇新文必须**：
- 链到 1 个支柱
- 链到 2 个地域/专题/行业
- 被 1–2 篇旧文反链

### 3.3 内容类型优先级

| 类型 | 示例 | 竞争 | 优先级 |
|------|------|------|--------|
| 金額系 | 50万・200万 ポートフォリオ | L | 高 |
| 比較系 | 商社 vs 銀行、電力 vs 通信 | L | 高 |
| 地域拡張 | 埼玉、千葉、兵庫、宮城 | L | 高 |
| 業界深化 | REIT、鉄道、食品、医薬品 | L–M | 中 |
| 税務系 | NISA 配当 非課税、源泉徴収 | M | 中 |
| シミュレーション | 100万円 月々配当 | L | 低 |

### 3.4 行业页释放顺序（与 seo-roadmap.json 一致）

```
P1: shosha-kouhaitou-2026 移除 noindex（触发条件满足后）
P2: bank-kouhaitou-2026（P1 后 14 天）
P3: denryoku-kouhaitou-2026、tsushin-kouhaitou-2026（P2 后 14 天）
P4: REIT、鉄道、食品、医薬品（91–180 天，按 trigger）
```

## 四、内链优化（持续）

### 4.1 首页

- 保持 2 条以上支柱页内链（高配当株ランキング、NISA おすすめ）
- 行业页上线后，增加对应行业入口

### 4.2 地域页

- 底部增加「関連地域」区块，链接 2–3 个邻近地域
- 链到 首都圏/関西/東海 等汇总页
- 链到 利回り・NISA 专题

### 4.3 专题页

- 每页至少 2 个地域 + 2 个行业/金额
- 文末「関連記事」统一 4–5 条

### 4.4 博客索引

- 按主题/地域分类展示
- 新文发布后 7 天内加入索引页

---

## 五、外链建设（第 4 月起）

### 5.1 目标

| 时间 | 外链目标 | 类型分布 |
|------|----------|----------|
| 第 4 月 | +5 | 数据型 2、客座 2、目录 1 |
| 第 5 月 | +10 | 数据型 2、客座 4、媒体 2、目录 2 |
| 第 6 月 | +15 | 持续同上，优先 dofollow |
| 第 12 月 | 累计 25–50 | 高质量为主 |

### 5.2 外链获取方式

| 方式 | 示例 | 质量 |
|------|------|------|
| データ型コンテンツ | 業種別利回りランキング、地域銘柄数 | 高 |
| ゲスト投稿 | 投資ブログ、地域メディア | 高 |
| メディア取材 | 統計提供、専門家コメント | 高 |
| ハブ・ディレクトリ | 国内金融系（合规） | 中 |
| パートナー | 証券 educational、金融教育 | 高 |

### 5.3 禁止行为

- 购买链接、PBN、垃圾站群
- 无关主题交换链接
- 短期内外链暴增
