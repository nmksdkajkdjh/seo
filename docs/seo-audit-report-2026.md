- 大部分图片有 alt 文本，且与主题相关

## 8. 总结

kouhaitou.jp 在技术 SEO 与内容结构上基础良好：sitemap、canonical、Schema 等配置到位，地域 + 专题 + 行业的内容矩阵已经成型。**未出现在搜索结果的主要原因是域名权重低、难以与大型财经媒体竞争主词**。

**提升路径**：

1. 聚焦长尾与细分，避免与日経等正面竞争
2. 强化内链网络与交叉链接
3. 有序释放 noindex 行业页
4. 补充金额系、比较系等新内容类型
5. 持续积累外链与品牌认知

**建议优先执行**：技术小修复、内链优化、商社页正式上线（移除 noindex 并加入 sitemap）。
