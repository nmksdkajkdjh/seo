## 2. オンページ最適化チェックリスト

### 全ページ共通

- [x] `<title>` に地域名＋メインキーワード＋2026
- [x] `meta description` 80〜120 文字、CTA 含む
- [x] `meta keywords` 地域＋高配当株,NISA,銘柄
- [x] `geo.region`（JP-13 東京等）
- [x] `geo.placename` 都道府県名
- [x] `canonical` URL
- [x] `hreflang="ja"`
- [ ] BreadcrumbList Schema
- [ ] FAQ Schema（該当記事のみ）

### 地域ページ特化

- [x] Article Schema + `about.Place`
- [ ] 見出し H2 に地域名＋キーワード自然挿入
- [x] 内部リンク：他地域ページ・ブログ一覧
- [ ] 地域別 CTA 文言（東京の皆様へ等）

## 3. 内部リンク構造

```
/ (index)
├── /blog/ (一覧)
│   ├── tokyo-kouhaitou-2026.html
│   ├── osaka-kouhaitou-2026.html
│   ├── aichi-kouhaitou-2026.html
│   ├── fukuoka-kouhaitou-2026.html  ← 新規
│   └── kanagawa-kouhaitou-2026.html ← 新規
├── 人気記事カード（6本）→ 各ブログ
└── その他の記事（4本）→ 各ブログ
```

**推奨**：index に「地域別おすすめ」リンクブロックを追加し、全地域ページへリンク。

## 4. 検索流入→LINE転換フロー

| 段階 | 施策 |
