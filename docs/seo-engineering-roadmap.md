# kouhaitou.jp 工程级 SEO 路线图

> 基于「搜不到」五大核心原因的实操路线图

---

## 一、搜不到的真正原因（核心）

| 原因 | 说明 | 本路线图对应措施 |
|------|------|------------------|
| ① 域名权重太低 | 新站 + 小站 → DA/DR 极低 → 无法排名 | 短期打好基础；中期扩大体量；长期外链积累 |
| ② 关键词竞争极度激烈 | 楽天・日経・東洋経済・マネックス等垄断主词 | 避开主词，聚焦长尾与细分 |
| ③ 内容体量太小 | 文章少、无 Topic Cluster | 中期扩至 30–50 篇 + 主题集群 |
| ④ 外链几乎为零 | 无外链 = 无权威度 | 长期 20–50 条高质量外链 |
| ⑤ 可能未被完全索引 | sitemap/robots/抓取延迟 | 短期：GSC 提交 + 索引验证 |

## 二、短期（1–2 周）：索引与基础

### 2.1 提交 Sitemap 到 Google Search Console

**当前状态**：
- `sitemap.xml` 已存在，39 URL
- `robots.txt` 正确引用 `Sitemap: https://kouhaitou.jp/sitemap.xml`

**执行**：
1. 登录 [Google Search Console](https://search.google.com/search-console)
2. 添加资源 `https://kouhaitou.jp`（若未添加）
3. 左侧「サイトマップ」→ 输入 `sitemap.xml` → 送信
4. 等待数日，检查「ページ」→「インデックス」确认收录数

### 2.2 修正 robots.txt（如需）

**当前**：
```
User-agent: *
Allow: /
Sitemap: https://kouhaitou.jp/sitemap.xml
```
无需修改，配置正确。

### 2.3 索引问题排查

| 检查项 | 路径 | 状态 |
|--------|------|------|
| sitemap 中有 search | `/search/` | 在 sitemap 中 → 建议移除（noindex 页无需提交） |
| noindex 页面 | shosha, bank, denryoku, tsushin, admin | 正确，不应被索引 |
| 主站 URL | https://kouhaitou.jp/ | 确保 301 重定向 www→非www 或反之统一 |

**建议**：从 sitemap 中移除 `/search/`，减少无效 URL。

### 2.4 增加内部链接

**可立即执行**：
- 首页「おすすめ資料」或主内容区增加 1–2 条指向 `2026-kouhaitou-kabuka-ranking`、`nisa-osusume-meigara-2026` 的锚文本
- 地域页底部增加「関連地域」区块，互相链接 2–3 个邻近地域
- 专题页文末「関連記事」统一为 4–5 条

### 2.5 提升页面加载速度

**可执行项**：
- 首屏 Hero 图片使用 `fetchpriority="high"`
- 评估 Tailwind 自托管，减少 CDN 依赖
- 确保图片 `loading="lazy"` 覆盖非首屏

### 3.2 建立 Topic Cluster（主题集群）

**核心主题**：高配当株

**集群结构**：
```
首页 (Pillar)
├── 地域集群：東京、大阪、愛知、福岡… + 埼玉、千葉、兵庫
├── 专题集群：NISA、利回り4%以上、減配しにくい、日経平均50
├── 行业集群：商社、銀行、電力、通信（按 roadmap 逐步解除 noindex）
└── 场景集群：100万円、50万円、塩漬け、暴落銘柄
```

**内链规则**：
- Pillar 页链到各集群 2–3 篇
- 集群内互相链接
- 每篇文章链回 Pillar 或核心专题 1–2 条

### 3.3 每周更新 1–2 篇

| 周 | 动作 |
|----|------|
| 1–2 | 新增 50万円・200万円 ポートフォリオ |
| 3–4 | 新增 埼玉・千葉 地域页 |
| 5–6 | 新增 兵庫・宮城 地域页 |

### 3.4 noindex 行业页释放顺序（参考现有 roadmap）

按 `site/data/seo-roadmap.json` 的 P1–P3 执行：
1. **P1**：首页主词 Top15 或利回り Top20 或 impressions 翻倍 → 商社页移除 noindex
2. **P2**：商社上线 14 天后、利回り Top15 → 銀行页移除 noindex
3. **P3**：P2 后 14 天 → 電力・通信移除 noindex

## 四、长期（3–6 个月）：外链与深度内容

### 4.1 获取 20–50 条高质量外链

**策略**：
| 类型 | 示例 | 优先级 |
|------|------|--------|
| 金融/投资媒体投稿 | 外部专栏、Guest Post | 高 |
| 地域商业/生活媒体 | 東京・大阪等地域媒体 | 中 |
| 目录/聚合站 | 日本国内合规目录 | 低 |
| 合作伙伴 | 券商、金融教育机构（合规范围内） | 高 |

**避免**：购买链接、垃圾站群、与主题无关的交换链接。

### 4.2 扩展到更多地域

**可追加**：埼玉、千葉、兵庫、宮城、新潟、栃木、群馬、長野 等

**URL 格式**：`/blog/{地域}-kouhaitou-2026.html`

### 4.3 发布深度文章（3000–6000 字）

**选题**：
- 高配当株 完全ガイド 2026
- NISA 成長投資枠 配当株 選び方
- 商社株 vs 銀行株 徹底比較
- 減配リスク 業界別 分析

**结构**：目录 + 小标题 + 数据/表格 + FAQ + CTA

## 七、重要提醒

1. **SEO 报告/路线图本身不改变排名**  
   本文件不会被执行引擎读取，不会直接提升权重。只有按此执行（提交 GSC、增加内容、获取外链等）才会生效。

2. **主词需长期积累**  
   「高配当株 おすすめ」等主词被巨头垄断，新站难以短期突破。优先做长尾与地域词。

3. **外链是瓶颈**  
   内容可自控，外链需外部合作。建议及早规划媒体投稿、合作伙伴等渠道。
