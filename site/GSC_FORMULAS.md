# GSC 周报 Excel 公式（直接可执行）

## 快速建表

1. 新建 Excel，命名 `GSC_WEEKLY_REPORT.xlsx`
2. 将 `GSC_WEEKLY_REPORT.csv` 内容粘贴到 Sheet1，或按下列结构建 3 个 Sheet
3. Sheet2 从第 2 行起填入公式后向下填充

---

## Sheet 1：Query 原始数据

| 列 | 字段 | 说明 |
|----|------|------|
| A | 日期 | 导出日期 |
| B | Query | 用户搜索词 |
| C | 页面 | 着陆页 URL |
| D | 展示 | 28天展示次数 |
| E | 点击 | 28天点击次数 |
| F | CTR | 点击/展示 |
| G | 平均排名 | 平均位置 |

**操作**：每周从 GSC 导出「搜索结果」→ 覆盖本表最近 28 天数据。

---

## Sheet 2：自动判断层

将 Sheet 1 数据复制到 Sheet 2，新增 3 列公式（从第 2 行起向下填充）：

### 1️⃣ 触发优化（Title）
```
=IF(AND(G2>=11,G2<=20,D2>=200),"改Title","")
```
- G = 平均排名，D = 展示

### 2️⃣ 触发优化（Description）
```
=IF(AND(G2>=5,G2<=10,F2<0.04),"改Description","")
```
- F = CTR（需为小数，如 4% = 0.04）

### 3️⃣ 不动
```
=IF(AND(G2<=5,F2>=0.08),"保持","")
```

### 4️⃣ 展示环比变化（7天）
假设 D9 = 7 天前同 query 的展示
```
=(D2-D9)/D9
```

### 5️⃣ 排名变化
```
=G9-G2
```
正数 = 排名上升 | 负数 = 排名下降

### 6️⃣ 优化优先级评分
```
=IF(AND(G2>=11,G2<=20,D2>=200),3,
 IF(AND(G2>=5,G2<=10,F2<0.04),2,
 IF(AND(G2>20,D2>=500),1,0)))
```
优先优化：3 → 2 → 1

## 操作流程

1. GSC 导出 CSV → 打开 `GSC_WEEKLY_REPORT.csv` 或新建 Excel
2. 粘贴到 Sheet 1
3. Sheet 2 复制 Sheet 1 数据，填入公式列
4. 筛选「改Title」「改Description」→ 按 A/B 库执行
